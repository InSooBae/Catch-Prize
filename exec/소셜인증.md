# 소셜 인증

- 네이버 소셜 인증
    - 네이버 어플리케이션 등록
    - yml 파일
    
    ```java
    security:
        oauth2:
          client:
            registration:
              naver:
                client-id: ${NAVER.CLIENT_ID}
                client-secret: ${NAVER.CLIENT_SECRET}
                redirect-uri: ${BASE.URL}${NAVER.REDIRECT_URI}
                authorization-grant-type: authorization_code
                scope:
                  - name
    
            provider:
              naver:
                authorization-uri: https://nid.naver.com/oauth2.0/authorize
                token-uri: https://nid.naver.com/oauth2.0/token
                user-info-uri: https://openapi.naver.com/v1/nid/me
                user-name-attribute: response
    ```
    
- 구글 소셜 인증
    - 구글 클라우드 콘솔 어플리케이션 등록
    - yml 파일
    
    ```java
    security:
        oauth2:
          client:
            registration:
              google:
                client-id: ${GOOGLE.CLIENT_ID}
                client-secret: ${GOOGLE.CLIENT_SECRET}
                redirect-uri: ${BASE.URL}${GOOGLE.REDIRECT_URI}
                scope:
                  - profile
    ```
    
- 카카오 소셜 인증
    - 카카오 어플리케이션 등록
    - yml 파일
    
    ```java
    security:
        oauth2:
          client:
            registration:
              kakao:
                client-id: ${KAKAO.CLIENT_ID}
                client-secret: ${KAKAO.CLIENT_SECRET}
                redirect-uri: ${BASE.URL}${KAKAO.REDIRECT_URI}
                authorization-grant-type: authorization_code
                client-authentication-method: POST
                client-name: Kakao
                scope:
                  - profile_nickname
    				provider:
              kakao:
                authorization-uri: https://kauth.kakao.com/oauth/authorize
                token-uri: https://kauth.kakao.com/oauth/token
                user-info-uri: https://kapi.kakao.com/v2/user/me
                user-name-attribute: id
    ```
